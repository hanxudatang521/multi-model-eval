# 模型效果 Markdown 对比工具

这是一个纯静态 HTML 工具，用来对比同一批 query 下多个模型的 Markdown 输出效果。

## CSV / XLSX 格式

第一行必须是表头：

```csv
query,模型A,模型B,模型C
```

从第二行开始：

- 第 1 列是 query。
- 第 2 列是第一个模型的具体效果。
- 第 3 列是第二个模型的具体效果。
- 如果继续增加更多列，也会被识别为更多模型输出。
- 模型输出可以是 Markdown，支持标题、列表、引用、代码块和表格等常见格式。
- XLSX 文件会默认读取第一个工作表。

## 使用方式

打开 `index.html` 会默认展示演示数据。点击左侧的“上传 CSV / XLSX 数据”，选择评测数据文件即可。页面会自动重建 query 列表和模型对比面板。

如果上传失败，请确认文件是 CSV 或 XLSX，且第一列是 query，第二、第三列是两个模型的具体效果。

点击“打乱 query”可以随机调整当前 query 列表的展示顺序。
