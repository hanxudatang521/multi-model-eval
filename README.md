# 模型效果 Markdown 对比工具

这是一个纯静态 HTML 工具，用来对比同一批 query 下多个模型的 Markdown 输出效果。

## CSV / XLSX 格式

第一行必须是表头：

```csv
query,工具参数,搜索结果,大模型回复A,大模型回复B
```

从第二行开始：

- 推荐有一列表头为 query。
- 推荐有两个模型输出列，列名可包含“大模型回复”“模型回复”“response”“output”等关键词。
- 其他参数列不会在评测页展示，但会在“导出评分”时原样保留。
- 模型输出可以是 Markdown，支持标题、列表、引用、代码块和表格等常见格式。
- XLSX 文件会默认读取第一个工作表。

## 使用方式

打开 `index.html` 会默认展示演示数据。点击左侧的“上传 CSV / XLSX 数据”，选择评测数据文件即可。页面会自动重建 query 列表和模型对比面板。

如果上传失败，请确认文件是 CSV 或 XLSX，且包含 query 和两个模型输出列。

点击“打乱 query”可以随机调整当前 query 列表的展示顺序。

点击“下载对比图”可以把当前 query 的前两个模型结果导出为一张 PNG 图片，图片会保留标题、加粗、列表、引用、代码块和表格等 Markdown 格式效果，方便分享评测案例。

点击“导出评分”可以下载一份 XLSX。导出的表会保留上传文件里的所有原始列，并在最后追加“评分结果”“胜出模型”“左侧模型列名”“右侧模型列名”“是否已评分”。
